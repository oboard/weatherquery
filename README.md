# oboard/weatherquery

一个使用 MoonBit 语言开发的天气查询工具，可以获取指定城市的实时天气信息。

## 功能特点

- 🌤️ 实时天气查询
- 📍 支持中国省份、城市、区县的精确天气查询
- 🌡️ 显示以下天气信息：
  - 温度
  - 天气状态
  - 风向和风力
  - 湿度
  - 气压

## 使用方法

1. 确保已安装 MoonBit 开发环境
2. 克隆本仓库
3. 运行程序：
```bash
moon run src/main/main.mbt --target native
```
或者
```bash
moon run src/main/main.mbt --target js
```
